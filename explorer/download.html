Run a Chakram Node

Download the app, open it, and you're mining in under 5 minutes. No command line needed.

⬇ Mac — Apple Silicon ⬇ Mac — Intel ⬇ Download for Windows Read the Docs
macOS — GUI App
1
Download the Mac app
Pick the right version for your Mac chip:

⬇ Apple Silicon (M1 / M2 / M3 / M4)   ⬇ Intel Mac

Not sure which chip you have? Click the Apple menu → About This Mac. If it says Apple M1 (or M2/M3/M4) download Apple Silicon. If it says Intel Core, download Intel.

It will save to your Downloads folder.
2
Unzip the file
Double-click Chakram-mac.zip in your Downloads folder. Mac will unzip it automatically.

You'll see Chakram.app appear right in your Downloads folder — that's the app.
Downloads folder
📦 Chakram-mac.zip
🖥 Chakram.app ← this is the app
3
Try to open Chakram.app
Double-click Chakram.app.

Mac will show a warning like this — this is normal for apps that aren't from the Mac App Store:
macOS security dialog
🚫 "Chakram" cannot be opened because Apple cannot check it for malicious software.

OK
Click OK to dismiss it — then go to the next step.
ℹ️ This happens because Chakram is open-source software distributed directly, not through the App Store. The app itself is safe.
4
Allow the app in System Settings
Open System Settings (the gear icon in your Dock or Apple menu).

Go to Privacy & Security.

Scroll down — you'll see a message saying "Chakram was blocked" with an Open Anyway button next to it.
System Settings → Privacy & Security
🛡 Privacy & Security

"Chakram" was blocked from use because
it is not from an identified developer.

Open Anyway
Click Open Anyway.
💡 On older Macs (macOS Ventura or earlier) this setting is in System Preferences → Security & Privacy → General tab.
5
Confirm and open
Mac will ask one more time to confirm. Click Open in the dialog.
Final confirmation dialog
macOS cannot verify the developer of "Chakram".
Are you sure you want to open it?

Cancel   Open
The app will launch. You only have to do this once — after this Mac remembers it's allowed.
6
Create your wallet
The first time you open Chakram, it will ask you to create a wallet.

Choose a password — this is used to protect your wallet file on this computer. Pick something you'll remember.

The app will show you a 12-word recovery phrase. Write these words down on paper and keep them somewhere safe. This is the only way to recover your wallet if you ever lose access to this computer.
⚠️ Never share your 12 words with anyone. Anyone who has them can take all your CHK.
7
Start mining
Once the wallet is set up, the app connects to the Chakram mainnet and starts syncing the blockchain. This takes a few seconds.

Click Start Mining to begin earning CHK. Your computer's CPU will work to solve puzzles — when you solve one you earn the block reward.
💡 Mining works on any modern Mac — you don't need special hardware. More CPU cores = more chances to win.

Linux — Command Line
1
Download the Linux binary
Go to the latest release and download chakram-linux, or use curl:
curl -L https://storage.googleapis.com/chakram-dist/latest/chakram-linux -o chakram
2
Make it executable
chmod +x chakram
3
Create a wallet
./chakram wallet new
Write down your 12-word recovery phrase somewhere safe.
4
Run a node and start mining
./chakram node --mine
To keep it running after you close the terminal, use screen or tmux:
screen -S chakram
./chakram node --mine

# Press Ctrl+A then D to detach. Node keeps running.
💡 For a permanent always-on node, see the docs for the systemd service setup.
